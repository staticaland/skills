# Staticaland Skills

Skills for Claude Code.

## Install

```
/plugin marketplace add staticaland/skills
```

## Skills

| Skill                | What it does                                             |
| -------------------- | -------------------------------------------------------- |
| **writing-revision** | Revises text to be clear, scannable, and action-oriented |
